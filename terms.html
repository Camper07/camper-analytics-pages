Camper Analytics - Terms of Service

Camper Analytics is a private analytics tool used only by the owner to track video statistics from TikTok and YouTube.

The tool does not allow public user registration and is not available as a public service.

The tool only reads video statistics for the TikTok account that authorizes access. It does not post, edit, delete, hide, or share content.

The data is used only for personal analytics and performance tracking.

Access can be revoked at any time from TikTok account settings.
